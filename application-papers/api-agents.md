
## API Agents
ユーザーが音声/テキストで指示するだけで専門的なソフトウェアを扱う業務を完了することを目指します。

既存のソフトウェアがREST APIやpython SDKに対応している場合、それはエージェントのツールになります。

製造業や化学の3Dモデリングなどの複雑なソフトウェアをエージェントが操作します。

日常の定型業務を代替するRPAの派生のAgentic Process Automationも含まれます。
